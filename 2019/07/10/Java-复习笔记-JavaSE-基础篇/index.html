1 包装类型的定义

1）八个基本类型对应有八个包装类，包装类是final 修饰的，无法被继承和重写的
2）包装类之间的比较: “==” 比较的是地址，在一定范围内是值比较, 以Integer为例， 比较源码如下：

Boolean, Character, Byte: < 127**
Short, Integer: **> -128 && < 127
