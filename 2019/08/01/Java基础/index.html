1 String

  • final 类型， 不可变
    不可变好处:
    (1) 可以做Hashmap 的key
    (2) 线程安全（StringBuffer 是线程安全的，StringBuilder 线程不安全）
  • java 不能隐式向下转型，这样会精度丢失

    2 继承

  • 访问权限
修饰符 类内部 本包 子类 外部包
public * * * *
protected * * * x
default * * x x
private * x x x
  • 子类重写父类的方法访问权限不得低于父类

3 抽象类和抽象方法

  • 抽象类不能被实例化

  • 接口成员和方法默认是public，不能是privated 和 protected, 字段默认是static final

  • 接口和抽象类使用选择
    (1) 使用接口

    • 不相关类实现同一方法
    • 多重继承

    (2) 使用抽象类

    • 相关类共享代码
    • 控制成员访问权限，不能全是public
    • 需要继承非静态和非常量字段

4 equals() 实现

  • 检查是否为同一个对象的引用，如果是返回true
  • 检查是否是同一的类型，如果不是返回false
  • 将Object 对象进行转型
  • 判断每个关键域是否相等
